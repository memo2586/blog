1. 几何着色器

渲染管线中，在顶点着色器与片段着色器之间，还有一个可以注入自定义的着色器：几何着色器。几何着色器从顶点着色器接受一个图元，可以在几何着色器中对输入的图元中的顶点进行自定义的变换后，重新装配为图元发送给片段着色器。

接下来要编写几何着色器，直接看一个例子：

  • 输入的图元类型：在几何着色器的顶部，在 in 关键字前使用 layout 修饰符声明从顶点着色器输入的图元类型，可以从顶点着色器接收下列任何一个图元值（括号内表示该类型图元需要的最小顶点数）：

    • points：绘制 GL_POINTS图元时（1）。
    • lines：绘制 GL_LINES 或 GL_LINE_STRIP 时（2）
    • lines_adjacency：GL_LINES_ADJACENCY 或 GL_LINE_STRIP_ADJACENCY（4）
    • triangles：GL_TRIANGLES、GL_TRIANGLE_STRIP 或 GL_TRIANGLE_FAN（3）
    • triangles_adjacency：GL_TRIANGLES_ADJACENCY 或 GL_TRIANGLE_STRIP_ADJACENCY（6）

  • 输出的图元类型：理所当然的我们也要在 out 关键字前使用 layout 修饰符声明几何着色器的输出图元类型，该修饰符接受下列图元值：

    • points
    • line_strip
    • triangle_strip

    同时，需要为 layout 中的 max_vertices 参数设置一个它最大能够输出的顶点数量，超过该参数大小的顶点都会被丢弃。

  • 输入变量：几何着色器的输入是由 GLSL 提供的内建变量，它被声明为一个接口块数组（一个图元可能会有多个顶点），大概就像是这样子：

  • 输出图元：每次调用 EmitVertex 时，gl_Position 中的向量值会被添加到待装配的图元中，在调用了一次或者多次 EmitVertex 后，调用 EndPrimitive，将当前待装配的图元装配后发送到片段着色器。当然，也可以调用多次 EndPrimitive 发送多个图元。
