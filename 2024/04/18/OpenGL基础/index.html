0. 环境配置（windows）

  • GLFW，最好自己编译，可以在官方下载页上找到，编译后链接 include（在源码中）以及编译生成的 glfw3.lib；

  • GLAD，可以在官方在线服务页面找到自己对应版本的库文件。链接 include 后记得将 glad.c 添加到工程文件中；

  • GLM，这是一个只包含头文件的库，可以在官方页面找到，直接将根目录下 ./glm 链接到include；

  • stb_image.h，将它加入到工程中，并创建 stb_image.cpp 输入以下代码：

  • Assimp，需要自己编译，编译完成后会得到 assimp-vcxxx-mt.lib 与 assimp-vcxxx-mt.dll 两个文件，链接 include（在源码文件中）以及 assimp-vcxxx-mt.lib，记得将动态链接库（.dll）添加到工程文件中；

    • debug：错误信息 -> 找不到 config.h -> 解决方法 -> config.h 在编译时产生直接去编译目录下找找并添加到 include

  • Dear ImGui（不是必须的但是用过都说好），在这里下载最新的发布版本并将根目录下的 .cpp / .h 文件（11个文件）以及 ./backends 目录下对应开发工具的文件（这里是 glfw + opengl3 共4个文件）整理好添加到你的工程文件中。
